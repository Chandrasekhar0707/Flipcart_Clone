# 🛒 Flipkart Clone Website

This project is a **Flipkart Clone Website** built using **HTML5 and CSS3**. It replicates the basic user interface of an e-commerce platform and is designed for learning and practice purposes in front-end web development.

---

## 📌 Features

* Responsive homepage layout
* Navigation bar with search box (UI only)
* Product sections and categories
* Banner and promotional sections
* Clean and structured design

---

## 🛠️ Technologies Used

* HTML5
* CSS3

---

## 🎯 Purpose

The main goal of this project is to improve front-end development skills by building a real-world UI clone. It helps in understanding layout design, styling, and structuring web pages like professional websites.

---

## ⚠️ Note

This is a **static website**. No backend functionality is included.
Features like login, cart, and payment systems are not implemented.

---

## 🚀 How to Use

1. Download or clone the repository
2. Open the project folder
3. Run the `index.html` file in your browser
